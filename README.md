# Chat app

This is a chat app.

`warning`: This app is under development.

# Set up

make sure you have :

- docker
- docker-compose
- git

clone this repo

```sh

git clone https://github.com/vivek80801/chat-app.git

```

then run

```sh
docker-compose up
```

This app should be runing on [localhost](http://localhost/ "Go to localhost")
